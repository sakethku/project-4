What's the Difference?

HTML
    Create The Structure
  • Controls the layout of the content
  • Provides the structure for webpage design
  • The fundamental building block of any web page

CSS
    Stylize the Website
  • Applies style to the web page elements
  • Targets various screen sizes to make webpage responsive
  • Primary handle the "look" and "feel" of a web page

Javascript
    Increase interactivity
  • Adds interactivity to a web pages
  • Handles complex functions and features
  • Programmatic code which enhances functionality
